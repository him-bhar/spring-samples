Getting Started A quick user guide

This site is useful for users who believe in the concept of contri-parties (parties funded by individual contribution). Usually what happens is one person volunteers to pay the complete bill and then takes up the task of distributing the individual shares and then sending reminder mails for payments. And the biggest trouble of all tracking the payments back.


This site makes all these tasks pretty easy to do. And does all the job of saving the individual shares, sending reminders and tracking payments.

Features Currently available

At the moment there are three available features in here.

  • Friends
  • Events
  • Shares

Friends:

This data is heart of the service you are trying to use. As you cannot distribute anything unless and until you do not have any friends added in your profile.

Events:

This is a good to have information, and is of use with the user of this application only. Benefit: While distributing the shares the user can map the current distribution with a event. Note: One event can be mapped to only one distribution.

Shares:

This is where the user will actually be doing the job of assigning shares to friends. Sending of alerts (payment reminders and payment confirmations) too is done from the shares section only.
